::accordion
  :::accordion-item{label="Which frameworks are supported?" icon="i-lucide-box"}
  QBox, QBCore and ESX. The framework is auto-detected on start — you don't configure it.
  :::

  :::accordion-item{label="Do I have to set the framework anywhere?" icon="i-lucide-cog"}
  No. Detection happens automatically. Just make sure your framework core starts **before** `alamo-market`.
  :::

  :::accordion-item{label="Which files can I edit?" icon="i-lucide-square-pen"}
  Two: `config.lua` (all gameplay settings) and `html/assets/css/variables.css` (colors). Both stay editable. Locale files in `locales/` are also editable.
  :::

  :::accordion-item{label="How do I change item prices for all 24/7 and LTD stores at once?" icon="i-lucide-circle-dollar-sign"}
  Edit the `convenience` catalog in `Config.ShopCatalogs`. Every location using that catalog updates automatically.
  :::

  :::accordion-item{label="How do I add a new shop location?" icon="i-lucide-map-pin"}
  Add an entry to `Config.ShopLocations` with a unique `id`, `coords`, and a `catalog`. See Shop Locations.
  :::

  :::accordion-item{label="Can a shop only allow card payments?" icon="i-lucide-credit-card"}
  Yes — add `paymentMethods = { 'card' }` to that location. See Payments.
  :::

  :::accordion-item{label="Can I lock a shop to a job?" icon="i-lucide-lock"}
  Yes — add `jobs = { 'police' }` (and optionally `minJobGrade`). See Job Restrictions.
  :::

  :::accordion-item{label="How do I change the UI color?" icon="i-lucide-palette"}
  Edit `variables.css`. Switch to a built-in preset (Yellow/Green/Blue/Red) or set your own `--accent` + `--accent-rgb`. See Color Scheme.
  :::

  :::accordion-item{label="How do I translate the UI?" icon="i-lucide-languages"}
  Edit the JSON in `locales/` or add a new file and point `Config.Locale` at it. See Localization.
  :::

  :::accordion-item{label="Is the money/item logic secure?" icon="i-lucide-shield-check"}
  Yes. The server re-validates shop access, payment method, distance, prices and amounts before charging. The client is never trusted.
  :::

  :::accordion-item{label="The player was charged but didn't get the item — why?" icon="i-lucide-triangle-alert"}
  The item `name` in the catalog must exist in your inventory system. A typo means `AddItem` fails. See Troubleshooting.
  :::
::
